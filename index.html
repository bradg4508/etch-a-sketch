Etch-a-Sketch

Click the "New Grid Size" button above to enter a grid size and start sketching. If you want to start over with your current grid, just click the "Reset Current Grid" button. You can go back and forth with how you want to color your grid by selecting either the "Classic Shade" button or the "Randomize Color" button. The default coloring setting for each new grid will be classic shading. Have fun and get to sketching!
